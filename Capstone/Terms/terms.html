Term Definition Example
Variable A container for a value that can be changed or reassigned.
Function A block of code that can be executed when called.
Array An ordered collection of values.
Object A collection of key-value pairs.
String A sequence of characters.
Number A numeric value.
Boolean A true or false value.
Operator A symbol that performs an operation on one or more values.
Control Flow The order in which code is executed.
Event An action or occurrence, such as a button click, that can trigger code execution.
Callback A function passed as an argument to another function, to be executed later.
